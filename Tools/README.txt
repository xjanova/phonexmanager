============================================
PhoneX Manager - Required Tools
by Xman Studio
============================================

Please download and place the following files in this folder:

1. adb.exe       - Android Debug Bridge
2. fastboot.exe  - Android Fastboot Tool
3. AdbWinApi.dll - ADB Windows API DLL
4. AdbWinUsbApi.dll - ADB Windows USB API DLL

Download from:
- Android SDK Platform Tools: https://developer.android.com/studio/releases/platform-tools

Or use the Setup tab in PhoneX Manager to auto-download.
