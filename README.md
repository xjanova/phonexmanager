# PhoneX Manager

**Universal Phone ROM Flash, Backup, Recovery and Engineering Tool**

By **Xman Studio**

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![.NET](https://img.shields.io/badge/.NET-8.0-purple)

## Features

### Device Management
- Auto-detect connected Android devices (ADB/Fastboot)
- Support for Qualcomm, MediaTek, Samsung, and Xiaomi devices
- Real-time device monitoring and status

### ROM Operations
- Full ROM backup with partition selection
- ROM flashing with multiple modes
- Boot image manipulation
- Partition management (GPT/MBR)

### Engineer Tools
- FRP Bypass (multiple methods)
- Screen Lock Bypass
- IMEI Read/Write/Repair
- Network Unlock
- Debloat (remove bloatware)
- Recovery Manager

### Utilities
- Built-in Hex Editor
- ADB Terminal
- Screen Mirroring (via Scrcpy)
- Logcat Viewer
- Checksum Calculator

### Setup & Drivers
- Auto-download Android Platform Tools (ADB/Fastboot)
- Driver installation (Google USB, Qualcomm, MTK, Samsung)
- Tool version management

## Requirements

- Windows 10/11 (64-bit)
- .NET 8.0 Runtime
- USB Debugging enabled on device

## Installation

1. Download the latest release
2. Extract to your preferred location
3. Run `PhoneXManager.exe`
4. Go to **Setup** tab to install required tools and drivers

## Usage

1. Enable USB Debugging on your Android device
2. Connect device via USB
3. Select device from dropdown
4. Use the various tabs for different operations
